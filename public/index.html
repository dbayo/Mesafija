API Mesafija

Init Subtext for header

Rest_lista Servicio que suministra el listado de restaurantes con sus datos básicos

Rest_datos Servicio que suministra el detalle de restaurante

Rest_disponibilidad_calendario Muestra los dias del calendario disponibles

Rest_disponibilidad_rango_plazas Servicio que suministra rango de comensales disponibles del restaurante

Rest_disponibilidad_turno_disponibles Servicio que suministra los turnos disponibles del restaurante. Si suministramos además el número de comensales, nos devolverá los horarios disponibles para la fecha y número de comensales seleccionados

Rest_disponibilidad_horas_disponibles Servicio que suministra las horas disponibles del restaurante

Rest_reserva_agregar Servicio que permite reservar mediante los datos proporcionados con los servicios rest_disponibilidad y usuario_datos

Rest_reserva_cancelar Servicio que permite cancelar una reserva mediante los datos proporcionados con el servicio usuario_datos

Usuario_login Servicio que permite el reconocimiento del usuario

Usuario_regpswd Servicio que permite procesar la regeneración de password del usuario

Usuario_registro Servicio que permite el registro del usuario

Usuario_datos Servicio que permite acceder a los datos de usuario

Usuario_editar Servicio que permite editar los datos de usuario mediante los datos proporcionados con el servicio usuario_datos

Valoracion Servicio que permite valorar un restaurante por el usuario mediante los datos proporcionados con el servicio usuario_datos

Usuario_favorito_agregar Servicio que permite marcar un restaurante como favorito por el usuario mediante los datos proporcionados con el servicio usuario_datos

Usuario_favorito_eliminar Servicio que permite desmarcar un restaurante como favorito por el usuario mediante los datos proporcionados con el servicio usuario_datos
